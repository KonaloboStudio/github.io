Skype 7

  • Skype 7 прекратила работу?

Официальная поддержка Skype 7 была завершена Microsoft в ноябре 2018, в январе 2019 года был заблокирован вход для версий 7.40, однако откат до более старой 7.36 решал ситуацию. Примерно в мае 2019 года старые версии вместе со своим протоколом были отключены полностью.

  • Есть ли способы вернуться к Skype 7?

К сожалению на момент июля 2024 метод восстановить старый официальный клиент не найден.

Моды Skype 8

  • Почему в модах заменяются только звуки и изображения?

В js файлах Skype 8 применена обфускация кода для улучшения безопасности и предотвращения взлома. Привести код в изначальное состояние очень сложно. Я не обладаю такими навыками чтобы полноценно изменить код новых версий.

  • Что за проблема с версией 8.49?

В старых клиентах Skype 8 не было возможности отвечать сообщением на фото. Поэтому когда какой то пользователь с новым клиентом Skype 8 использует эту функцию, старые версии не могут правильно обработать такие сообщения и вылетают в синий экран.

  • При попытке запустить установщик открыватеся окно Windows SmartScreen

Проект создается за бесплатно на чистом энтузиазме, следовательно у нас нету возможности использовать цифровые подписи для установщиков. А из за их отсутствия Windows SmartScreen может выдавать уведомления о подозрительном файле.

  • Почему при установке на несколько миллисекунд открылась командная строка?

В этот момент установщик блокирует автообновление Skype и регистрирует копию Skype в "программы по умолчанию". Вы всегда можете проверить файл antiupd_protocol_patch.bat в папке установки Skype и убедиться что установщик больше ничего не делает с вашим устройством.

Группы

  • Сколько времени нужно ждать, пока моя группа будет добавлена в список?

Обычно на это уходит до нескольких дней. Однако иногда я могу быть слишком занят в реальной жизни или другими проектами чтобы быстро добавить группу в список.

  • Какие требования для добавления группы в список?

Для добавления группы в список в ней должно быть МИНИМУМ 10 участников и хотя бы 25 сообщений за неделю. В ее названии не должно быть нецензурной лексики, а в самой группе не должно быть противозаконных действий или разжигания ненависти.

  • Могу ли я установить правила для своей группы?

Конечно, в Google форме есть возможность добавить правила, которые будут отображаться при попытке подключиться к ней.

Skype Web Wrapper

  • Что такое Skype Web Wrapper?

Skype Web Wrapper - это старая версия Skype For Web которая обладала интерфейсом, схожим с Skype 7.

  • Что означают названия ReSkype или TrySkype?

Это названия клиентов на базе Skype Web Wrapper которые были популярны летом 2023 года.

  • В некоторых чатах я вижу упоминание Skype Classic, что это такое?

В большинстве случаев если в теме разговора рядом упоминаются Skype 4.0, Electron, app.asar или Skype Classic то вероятно речь идет про один из Skype Web Wrapper клиенте.

Моды Skype для Android

  • При попытке войти в аккаунт на Skype 7.х, 8.х у меня бесконечная загрузка. У меня Android 4.х.

В 2021 году для старых версий Android истекли действия сертификатов. Именно из за этого перестали работать некоторые сайты в том числе и домены Microsoft для входа в аккаунт. Вам необходимо найти новые сертификаты и обновить их для восстановления работы клиента.

  • Почему последняя версия мода - 8.72?

В более новых версиях Skype для Android было сильно изменено строение ресурсов, а также добавлена проверка хешей файлов. Обойти это можно но это займет немного больше времени...

  • У меня Skype 7.48. Перестали загружаться фото и другие данные

Обновитесь до Skype Lite 1.88 или Skype 8.15, у них еще старый дизайн.
