For successful adventures on older versions, you need to have a high level, a lot of diamonds and coins. We have created saves with such conditions. The only negative: the presence of ROOT rights on your device (the installation method for non-ROOT devices will be ready later)

 

Installation:

  • Download the save file for the required version
  • Transfer the file com.pixel.gun3d.xml (com.pixel.gun3d.v2.playerprefs.xml on 10.3+) to /data/data/com.pixel.gun3d/shared_prefs
