Mods of the old Web Client

Here you can try two key web client mods. Both are under development, so some features may be missing.
   TrySkype

The first public client built on the Linux Web Wrapper client. It is under development, has its own servers, from where the necessary scripts are already loaded.
Download (Discord Server)
Download (TrySkype Website)
   Classic Skype by vl0d

Based on Skype for Linux Beta (Skype Web Wrapper) and it has the UI of Old Skype for Web (2017).
It's in constant developement and unlike TrySkype it connects only to the official Microsoft servers, the modifications are only done client-side.
Next release will feature working incoming calls, settings will save without the FlagServer and much more.
Download (Discord Server)
