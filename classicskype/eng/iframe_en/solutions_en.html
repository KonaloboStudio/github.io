Method 2 - Viewing the Interface with Window in Process Hacker 2

The method is to simply enable the windows we need through the Process Hacker 2 program

 

1. Launch Skype. This trick can be done with versions 1.x / 2.x / 3.x / 5.x / 6.x / 7.x

4.x IS NOT SUPPORTED!!!

2. On the login window, open Process Hacker 2, look for the Skype process

3. RMB on the desired process -> miscellaneous -> windows

4. In the menu that opens, we simply experiment, including different window states by pressing RMB -> visible
