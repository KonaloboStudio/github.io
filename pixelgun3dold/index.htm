Для успешных похождений на старых версиях нужно иметь высокий уровень, много алмазов и монет. Мы создали сохранения с такими условиями. Единственный минус: наличие ROOT прав на вашем устройстве (способ установки для не-ROOT устройств будет готов позже)

 

Установка:

  • Скачать сейв для нужной версии
  • Файл com.pixel.gun3d.xml (com.pixel.gun3d.v2.playerprefs.xml на 10.3+) перекинуть в /data/data/com.pixel.gun3d/shared_prefs
