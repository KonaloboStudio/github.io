Installing capes and skins

- On versions 7.1.1 and below:
You just need that the weight of the cloak and skin does not exceed 1 mb (otherwise there will be a reconnect to the servers during the match) + be sure that the skin is cropped in 64x32 format

Take change the name of your skin to Mult_skin_0 and cape to cape_CustomTexture by removing the postscript .png/.jpg

Now we just transfer these files to Android/Data/com.pixel.gun3d/files and replace them with the existing ones

- On versions From 8.0.0 to 10.2.1

Unfortunately, skins without root cannot be installed on them. But the cloak is possible - you will need to close it when you first start the game during loading (without waiting for the end (so that the game date folder appears) and install the cloak in the same process as on 7.1.1.

How to install games with cache on Android?

Step 1. Download the desired APK file and its cache.
Step 2. Open any file manager, find the downloaded APK in the Download folder and open it to install the application. After installation, do not launch the application - go directly to the next step.
Step 3. Most often, the cache is distributed as an archive. If this is the case, find the downloaded cache in the file manager and unzip its contents (it should be a folder like com.pixel.gun3d with an OBB file inside) along the Android/obb path.
Step 4. After the unpacked folder appears in the Android / obb directory, you can run the installed game.
