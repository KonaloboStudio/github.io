TrySkype
The first public client that based on Skype Web Wrapper (Skype for Linux Beta). It is under development, has its own servers, from where the necessary scripts are already loaded. Can't call but it will be soon.
Join their Discord Server
Download (Fieme Blog)
ReSkype
ReSkype (Revived Skype) is a modification for Skype Web Wrapper (Skype for Linux Beta) that attempts to make the client look as close as possible to Skype Classic and restore all possible functionality. ReSkype connects only to the official Microsoft servers, has fixed a lot of features (messaging, calls, etc), and is ready for day-to-day usage. Screen sharing support is coming soon as well
