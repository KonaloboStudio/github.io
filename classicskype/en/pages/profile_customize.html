HONORARY USER SYSTEM

 

Honorary users are people who take part in the development of the site (searching for bugs, submitting ideas and suggestions, and generally promoting skype-related topics).

To become an honorary user, you only need to help the project once (the methods are described just above)

 

They have a number of advantages:

 

  • Orange profile by default (in case the user does not change the profile background)

 

  • Honorary users are located above regular users in the list

 

  • Honorary users can add their own background to the profile, the location of some elements

 

EDITING YOUR PROFILE BACKGROUND

 

  • Honorary users can also change the background of their profile as well as the location of some elements
  • To change the background you need to contact KonaloboStudio (live:gs54390)
  • You can change the background once every 2 weeks
  •  

PROFILE BACKGROUND MUST COMPLY WITH GITHUB AND SKYPE CONTENT PUBLICATION RULES
